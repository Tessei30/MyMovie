# My Movie

"My Movie"は映画の感想を投稿したり、今まで見た映画の記録に使用できるアプリケーションです。
 
# Features
 
"My Movie"ではログイン、ログアウト、映画の感想の投稿、編集、検索機能、いいね機能等が実装されています。
 
# Usage
 
```bash
git clone https://github.com/Tessei30/MyMovie
cd mymovie
bundle install
rails db:create
rails db:migrate
rails s
```
